# Currency Converter Pro

Графическое приложение для конвертации валют с использованием реальных рыночных данных.

**Автор:** Воинов Никита

## Требования (Зависимости)
Для работы приложения необходимы:
- **Python 3.x**
- **Tkinter** (обычно поставляется с Python)
- **Requests** (библиотека для сетевых запросов)

## Установка и запуск
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Установите зависимости:
   ```bash
   pip install requests
   ```
3. Зарегистрируйтесь на [ExchangeRate-API](https://exchangerate-api.com) и получите бесплатный ключ.
4. Вставьте ваш ключ в файл `main.py` в переменную `self.api_key`.
5. Запустите приложение:
   ```bash
   python main.py
   ```

## Использование и тестирование
- **Валидация:** Попробуйте ввести текст или отрицательное число — приложение выдаст предупреждение.
- **История:** Все расчеты автоматически сохраняются в `history.json` и отображаются в таблице при следующем запуске.
- **Ошибки:** При отсутствии интернета или неверном API-ключе программа выведет подробное описание проблемы.
