# Currency Converter

Приложение для конвертации валют в реальном времени.

**Автор:** Никита Воинов

## Инструкция по получению API-ключа
1. Перейдите на сайт [exchangerate-api.com](https://exchangerate-api.com).
2. Введите свой email и нажмите "Get Free Key".
3. Подтвердите почту и скопируйте API Key из личного кабинета.
4. Вставьте ключ в переменную `API_KEY` в файле `main.py`.

## Использование
1. Запустите приложение: `python main.py`.
2. Введите сумму (например, `100`).
3. Выберите валюты (например, USD -> RUB).
4. Нажмите "Конвертировать". Результат отобразится в окне и сохранится в таблицу истории.
