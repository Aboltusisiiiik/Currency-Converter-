# Movie Library

Приложение для ведения личной кинотеки. Позволяет сохранять список просмотренных фильмов, их жанры, года выпуска и личный рейтинг.

**Автор:** Воинов Никита

## Функциональные возможности
- Добавление фильмов с валидацией (год — число, рейтинг — от 0 до 10).
- Автоматическое сохранение базы данных в формат JSON.
- Фильтрация списка по жанрам и конкретному году выпуска.

## Требования
- Python 3.x
- Библиотека Tkinter (стандартная)

## Запуск приложения
1. Склонируйте репозиторий.
2. Запустите файл: `python movie_library.py`.

## Примеры тестов (Использование)
1. **Позитивный тест:** Введите "Интерстеллар", Фантастика, 2014, 9.5. Нажмите добавить. Фильм появится в таблице.
2. **Негативный тест:** Введите рейтинг "11" или год "текст". Приложение выдаст ошибку валидации.
3. **Фильтрация:** Выберите жанр "Драма" и нажмите "Применить". В таблице останутся только драмы.
